# gsh — 一个用 Go 写的迷你命令行 Shell

这个项目是一个使用 Golang 编写的简单命令行工具。
它能读取用户输入、执行系统命令（如 pwd、ls、cd），并支持退出功能。
换句话说，它是一个轻量级的“Go 版终端”。

⸻

#### 🧭 项目结构

```
gsh/
├── cmd/
│   └── main.go         # 主入口：负责读取用户输入
├── internal/
│   └── exec.go/        # 核心逻辑：解析并执行命令 
└── go.mod              # 模块定义文件

```

这里采用了 Go 官方推荐的项目结构：

• cmd/：用于放主程序入口；

• internal/：放业务逻辑模块（不会被外部引用）；

• go.mod：定义模块名（建议命名为 gsh）。

⸻

#### 🚀 如何运行
首先初始化模块：

```go mod init gsh```

然后执行主程序：

```go run ./cmd/main.go```

看到提示后输入命令即可，例如：

> pwd
> /Users/demo/Desktop
> cd ..
> ls
> exit

💡 提示：exit 命令会退出程序，相当于在终端按 Ctrl + C。

⸻

#### ⚙️ 功能说明
1. 支持执行系统命令
•   直接输入 Linux / macOS 命令即可执行，比如：

```sh
ls
pwd
echo hello

```

2. 支持 cd 切换目录
•  示例：```cd /```


    •  若路径不存在，会提示：

```路径不存在```

3. 支持退出命令
•  输入：```exit```


    •  立即退出程序。

⸻




⚠️ 注意：
1.  本示例假设你在 Unix 系统（macOS 或 Linux）上运行。
如果在 Windows 上执行，部分命令（如 ls、pwd）可能无法使用。
2.  程序在执行命令时，直接继承宿主机的标准输入输出，
因此执行结果会直接显示在当前终端中。
